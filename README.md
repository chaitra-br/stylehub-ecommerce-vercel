🛍️ StyleHub – Modern E-commerce Frontend

StyleHub is a modern, responsive e-commerce frontend application built using React.
The project focuses on clean UI, smooth user experience, and core e-commerce functionality such as product browsing, size selection, cart management, and checkout flow (UI).

This project was built as part of my frontend portfolio to demonstrate real-world UI/UX patterns and state management.

✨ Features

🏠 Clean, modern homepage with featured sections

👕 Product listing pages (Men / Women / Kids)

🔍 Product details page with:

Image gallery with thumbnail selection

Size selection (mandatory before adding to cart)

Interactive “Add to Cart” feedback

🛒 Shopping cart:

Shows selected size per product

Quantity & total price calculation

Order summary section

🔐 Login / Signup UI (frontend only)

🌙 Dark-themed, minimal, modern UI

📱 Fully responsive (desktop & mobile)

⚡ Smooth micro-interactions for better UX

🛠️ Tech Stack

React.js

React Context API (state management)

React Router DOM

CSS (custom, no UI libraries)

Vercel (deployment)

🧪 Getting Started Locally

Follow these steps to run the project on your local machine.

1️⃣ Clone the repository
git clone https://github.com/your-username/stylehub-ecommerce.git
cd stylehub-ecommerce

2️⃣ Install dependencies
npm install

3️⃣ Start the development server
npm start


The app will run at:
👉 http://localhost:3000
